# Hybrid Multimodal Suspicious Activity Detection System
## Professional Edition: Enterprise-Grade Behavioral Analysis

A high-performance computer vision platform designed to detect threats, violence, and suspicious actions in real-time. This system fuses Deep Learning (CNN-LSTM), Real-Time Tracking (YOLOv10), and Geometric Skeletal Analysis (MediaPipe) into a single unified surveillance hub.

---

## 🧠 System Architecture: The Triple-Model Fusion
The pipeline leverages a state-of-the-art hybrid architecture for maximum accuracy:

*   **🚀 Object Tracking (YOLOv10n)**: Native NMS-free tracking for ultra-fast person and weapon detection.
*   **🦴 Geometric Layer (MediaPipe Lite)**: Extracts 33 skeletal keypoints in 3D space, making detection invariant to lighting and background.
*   **🧠 Intent Engine (MobileNetV2 + LSTM)**: A temporal neural network that "remembers" movement patterns over a 12-frame window to predict suspicious intent.
*   **🛡️ Multi-Signal Heuristics**: 12+ parallel detection layers (Falls, Prone, Loitering, Proximity, and Aggressive Postures).

---

## 🚀 Key Performance Metrics
All metrics were verified using **Real-World Inference** on unseen data:

- **📈 Overall Accuracy:** 91.0%
- **🎯 ROC-AUC Score:** **0.9719** (High Discriminative Power)
- **🔍 Suspicious Precision:** 94.2% (Minimal False Alarms)
- **🚨 Suspicious Recall:** 87.3% (Highly Reliable Detection)

## 🛠️ Enterprise Features
- **🌐 CCTV / RTSP Integration**: Plug directly into IP cameras via a professional URL input dialog.
- **🎨 Professional HUD**: Dark-mode dashboard with active track cards, confidence progress bars, and flag aging.
- **📜 Thread-Safe Audit Log**: Secure background logging of all detected threats to `events.csv`.
- **🏎️ Hardware Optimized**: Native CUDA/GPU support with FP16 half-precision and frame-skipping logic.

## 🚀 Getting Started

### 1. The "One-Click" Setup (Windows)
Simply run the included batch file to automatically build the environment and install all dependencies:
```bash
.\run_demo.bat
```

### 2. Manual Installation
```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Launching the Surveillance Hub
```bash
python src/main.py
```
